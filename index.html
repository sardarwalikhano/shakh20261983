بەخێربێیت بۆ ماڵپەڕی فەرمی شاخ دیلیڤەری (ShakhDelivery). دۆمەینەکەت بە سەرکەوتوویی لەگەڵ GitHub و Vercel بەستراوە. لەبەرئەوەی ئەم پڕۆژەیە ئەپڵیکەیشنێکی مۆبایلە (سیستەمی Android)، لەم ماڵپەڕە دەتوانیت ئەپەکە دابەزێنیت و داتابەیسی سوپابەیسەکەت ڕێکبخەیت!

لەبارەی ئەندرۆید ئەپەکەوە

ئەم پڕۆژەیە بە وەشانی مۆدێرنی Jetpack Compose و زمانەکەی بە Java/Kotlin کراوە. ئەپێکی تەواو سەربەخۆیە بۆ کڕینی خواردن لە چێشتخانەکانی هەولێر بە سیستەمی دیلیڤەری خێرا. تایبەتمەندییە ناوازەکانی:

  • ✔ زمانەکانی کوردی (Kurdish) و ئینگلیزی (English)
  • ✔ دزین و وێنەگرتنی جی پی ئێسی مۆبایلەکە بە شێوەی لایڤ
  • ✔ پەیوەستبوون بە داتابێسی لایڤی سوپابەیس (Supabase API)
  • ✔ دواندنی ستۆری کاتی (Stories UI) لە چێشتخانەکانەوە
🚀

About the Android App

ShakhDelivery is engineered purely with modern **Jetpack Compose** and **Kotlin** on the Android framework. It is integrated natively with Supabase endpoints that sync live menu items, stories and delivery trackers.

Device SDK Support: Android API 24 - 34+
UI Library: Material Design 3 (M3)

⚙️ هەنگاوەکانی تێکەڵکردنی داتابێس (Supabase Live Setup)

بۆ پەیوەستکردنی داتابێسەکە، تکایە بچۆ ناو پۆرتاڵەکەی Supabase (supabase.com) و کۆدەکەی خوارەوە لەبەشەکە کۆپی بکە و پاشان ڕەن بکە لەبەشی SQL Editor بۆ ئەوەی خشتەکانی چێشتخانە، مینۆ، مێز، ئۆردەر و ستۆرییەکان دروست ببن:

SUPABASE_SEED.sql

✔ کۆدەکە کۆپی کرا! ئێستا دەتوانیت لە سوپابەیس ڕەنی بکەیت.

🔑 چۆن کلیلەکان لە ئەپەکەم تێکەڵ بکەم؟

  1. 1 ئەندرۆید ئەپەکە لە ناو مۆبایلەکەت بکەرەوە.
  2. 2 لە سەرەوە لای ڕاست کرتە لەسەر نیشانەی ڕێکخستنەکان (Settings Gear ⚙️) بکە.
  3. 3 لەوێ ناونیشانی داتابێسەکەت (Supabase URL) و کلیلەکەت (Supabase Anon Key) بنووسە.
  4. 4 کرتە لەسەر پاشکەوتکردن بکه داتابێسەکە دەبێتە لایڤ بۆ هەموو مۆبایلەکان!
